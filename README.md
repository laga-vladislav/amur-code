# Описание продукта
Наше решение в текущем виде включает в себя генератор оглавления, макетов и слайдов на основе созданного пользователем макета, либо на основе выбранного стиля повествования.
Также есть редактор, позволяющий изменять сгенерированные слайды вручную, либо с помощью нейронной сети.

# Инструкция по развертыванию в локальном контуре
Скопировать `.env.example` в файл `.env`, отредактировав токен для взаимодействия с внешними API РТК.
